CHIP-8 SPECIFICATION SUMMARY:

CHIP-8 is an interpreted programming language, first used on COSMAC VIP and Telmac 1800 8-bit microcomputers in the mid-1970s. It features 16 8-bit registers (V0-VF), an index register (I), a delay timer (DT) for timing operations, a sound timer (ST) for generating a mono beep, and a stack for subroutine nesting.

OPERATIONAL MAPPINGS:
Register VF acts as a carry flag in addition, a borrow flag in subtraction, and a pixel collision detector during screen draw instructions.
Timer frequencies are set to decrements of exactly 60Hz.
